Debian

O Debian é um sistema operacional (SO) livre para seu computador. Um sistema operacional é um conjunto de programas básicos e utilitários que fazem seu computador funcionar.


O Debian é mais que um simples SO: ele vem com mais de 59000 pacotes contendo softwares pré-compilados e distribuídos em um bom formato, que torna fácil a instalação deles na sua máquina.


Os sistemas Debian atualmente usam o kernel Linux ou o kernel FreeBSD. O Linux é parte de um software criado inicialmente por Linus Torvalds com a ajuda de milhares de programadores(as) espalhados por todo o mundo. O FreeBSD é um sistema operacional que inclui um kernel e outros softwares.


No entanto, existe um esforço para fornecer o Debian para outros kernels, principalmente para o Hurd. O Hurd é um conjunto de servidores que rodam em cima de um microkernel (como o Mach) a fim de implementar diferentes características. O Hurd é um software livre produzido pelo projeto GNU.
